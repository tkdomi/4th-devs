# User Preferences

## Output format
- Structured markdown with clear section headers
- Use checkboxes for action items: `- [ ]`
- Date format: ISO 8601 or YYYY-MM-DD

## Communication style
- Concise, action-oriented
- No fluff. Lead with what matters.
- Bullet points over paragraphs

## Priority rules
- **Always surface overdue tasks first** — they go at top of Action Items
- Escalate anything skipped 2+ days — move to Escalated section with (Nd) suffix
- Blocked items: note blocker, don't treat as urgent unless dependency is resolved

## Escalation note
Escalate anything skipped 2+ days. If it appeared in yesterday's note and wasn't done, increase priority and flag in Escalated.
